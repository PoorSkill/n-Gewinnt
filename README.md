# n-Gewinnt
n-Gewinnt Java Spiel um die Langeweile in GBI zu bekämpfen 
*Random Lerngruppe: Anton, Florian, Jan und Merlin*


**Regeln**
Jeder Spieler platziert nacheinander einen Stein auf dem Spielfeld. Es kann nur die x-Position der Steine gewählt werden, da die Steine auf den untersten freien Platz *fallen*. Der Spieler, welcher es als erstes schafft, eine horizontale oder senkrecht Reihe aus seinen Steinen, von der gewählen Anzahl von Punkten, zu setzten, gewinnt. 

**Features:**
- Einstellung der Spielfeldgröße
- Einstellung der benötigten Punkte, welche zum Sieg benötigt werden
    
**Noch kommende Features:**
- Verschiedene Sprachen (Deutsch/Englisch)
- Einzelspieler, Dou und bis zu n-Spielern
- Diagonales gewinnen
